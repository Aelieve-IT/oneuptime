# Some basic commands for Clickhouse

## Show tables in the database
    
```sql  
show tables from oneuptime
```

## Show table structure

```sql
desc table_name from oneuptime
```

## Show table data

```sql
select * from table_name
```

## Delete table data

```sql
truncate table_name
```

## Delete table

```sql
drop table oneuptime.table_name 
```
